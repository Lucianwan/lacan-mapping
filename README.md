# lacan-mapping
这是一个致力于将拉康派精神分析（Lacanian Psychoanalysis）临床流程与核心理论进行系统化、可视化以及开源化展示的 GitHub 项目。  我们深知拉康著作那晦涩、如“巴洛克式”复杂的巴别塔风格——大量的长难句、新词汇与迷宫般的逻辑，常常让初学者在文字的浇筑中迷失方向。为了打破科学实证主义对个体独特欲望的束缚，传递精神分析的终极伦理和个体差异，本项旨在通过结构化的开源代码、清晰的流程图解以及文本分析工具，为每一位对精神分析感兴趣的“主体”提供一个人工肺（Artificial Lung），在拉康那不可呼吸的深刻思想中，为你撑开一片呼吸的例外之地。
